Habit++

习惯清单

  • 每个用户可以管理自己的习惯清单，并且通过可以加入小组，获得小组习惯
  • 系统会根据用户设定的频率，将待完成的习惯放到用户的Todo List中
  • 用户可以根据自己实际情况设置完成进度

监督&回报机制

  • 积分制度，根据习惯难易程度用户可获得一定积分，未完成则会扣除一定积分
  • 小组监督，小组成员未完成小组习惯，全队成员扣除少量积分
  • 排名系统，好友排名，小组排名

记录&追踪

  • 对用户历史数据进行可视化
  • 用户可查看个人习惯历史完成情况

用户信息

  • 用户名（username，唯一）
  • 电子邮箱（email，唯一）
  • 密码（password，Hash值）
  • 性别（gender，默认保密）
  • 昵称（nickname）
  • 头像（avatar，路径）
  • 出生日期（birthday）
  • 用户角色（role，管理员？）
  • 总分数（score）

小组信息

  • 小组ID（id，唯一）
  • 小组名称（name）
  • 创建者（creator）

习惯信息

  • 习惯ID（id，唯一）
  • 习惯名称（name）
  • 是否是小组习惯（is_team_habit，Boolean）
  • 创建者（creator，用户 / 小组）
  • 难易程度（difficulty，简单 / 中等 / 困难）
  • 执行频率（frequency，一天一次 / 工作时间每小时一次 / ...）

习惯进度信息

  • 用户名（username）
  • 习惯编号（habit_id）
  • 更新时间（time，日期时间类型）
  • 已完成次数（combo）
  • 完成状态（status，0 - 100%）

分数变动信息

  • 用户（user）
  • 更新时间（time，日期时间类型）
  • 变动情况（delta_score，有符号整数）
  • 变动原因（reason）

作为联系的信息

  • 用户ID（user1）
  • 好友ID（user2）

好友关系信息

小组成员信息

  • 小组ID（team_id）
  • 成员ID（member）
